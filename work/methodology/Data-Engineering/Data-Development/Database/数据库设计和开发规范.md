## 查询使用规范

查表规范
1. SELECT 查询中，字段转换后必须使用新的别名，禁止与原字段名重复，产生歧义
2. SELECT 查询中，除非是上下游是强耦合，否则禁止使用 `SELECT *` 查询，而使用具体查询字段

# 大数据主流技术栈地图


所有大数据技术，主要就是用于解决围绕着大数据的六大问题，**采集、处理、传输、存储、分析、管理**。

目前从大数据相关技术的发展路线来看，未来的“数据采集”、“数据处理”（流处理、批处理）、“数据分析”（OLAP），三类技术会合并为一个（如字节的 Flink 流批一体和 Flink OLAP），个人理解，因为这三类技术，都和分布式计算深度相关，因此未来大概率会合并。而“数据存储”、“数据传输”和“数据管理”，则会独立出来。

# 探索性数据分析 (EDA) 基础教程


数据探查，又名为探索性数据分析（Exploratory Data Analysis，EDA）。实际上就是在进行数据开发和深度分析之前对数据集进行初步探索和理解的过程，是一种简单的数据分析活动，用于发现数据中最明显的问题。

[IBM - What is exploratory data analysis?](https://www.ibm.com/topics/exploratory-data-analysis)

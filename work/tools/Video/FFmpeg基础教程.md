# FFmpeg基础教程


## 安装

### Linux

**Ubuntu**
```bash
apt-get update
apt-get install ffmpeg
```

**RHEL**
```
yum makecache
yum install ffmpeg
```


## 常用命令

### Syntax
```bash
ffmpeg [global_options] {[input_file_options] -i input_url} ... {[output_file_options] output_url} ...
```

# Git 可视化工具基础教程


## VSCode Gitlens

VSCode Plugin 直接安装即可：
https://github.com/gitkraken/vscode-gitlens
特点：
1. 开源
2. 功能丰富，支持多 Repository 管理
配置教程：
1. VSCode 创建 Workspace
2. VSCode 中添加文件夹到当前 Workspace
3. GitLens 即可自动识别这些文件夹下的首个 Git 仓库

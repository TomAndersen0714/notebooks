**下载地址：**

- CentOS官方下载地址：https://www.centos.org/download/
- 阿里云镜像站：https://mirrors.aliyun.com/centos/

本人选择的阿里云镜像站的CentOS IOS镜像文件，版本为[CentOS-7-x86_64-DVD-2009](https://mirrors.aliyun.com/centos/7.9.2009/isos/x86_64/)

# Python 开发规范

## 环境

【强制】每次引入和新建 Python 项目时，都必须创建对应的虚拟环境，进行环境隔离。[Python虚拟环境基础教程](work/programming/Python/Python虚拟环境基础教程.md)

## 编码

【推荐】不建议使用 relative imports，建议统一使用 absolute imports，避免两者混合使用，不利于阅读和理解。

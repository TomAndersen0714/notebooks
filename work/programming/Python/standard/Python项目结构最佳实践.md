# Python 项目结构最佳实践

Python Library 项目

```bash
<project_name>
├── LICENSE
├── README.md
├── docs
├── poetry.lock
├── <python_package_name>
├───├──__init__.py
├───├──main.py
├── pyproject.toml
├── tests
```

客户端项目：

```bash
<project_name>
├── LICENSE
├── README.md
├── docs
├── poetry.lock
├── src
├───├──__init__.py
├───├──main.py
├── pyproject.toml
├── tests
```

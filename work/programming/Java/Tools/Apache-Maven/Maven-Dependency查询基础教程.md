## 通过 IDEA 检索

在项目的 pom.xml 文件中，右键后打开菜单，点击 Generate

![](resources/images/Pasted%20image%2020240913145439.png)


在自动生成菜单中，选择 Dependency，即可支持检索已经缓存的 Maven Repository 中的 Dependency。

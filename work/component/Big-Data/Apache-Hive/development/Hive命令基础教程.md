# Hive 命令基础教程

Hive 命令除了 Hive SQL 之外，还包含有 HDFS、Hive Config 等相关子命令。

Hive 命令处了能在 Hive Shell 中执行，同样也能在各种通过 JDBC 连接的客户端脚本中执行，如 IDEA、DataGrip 等。

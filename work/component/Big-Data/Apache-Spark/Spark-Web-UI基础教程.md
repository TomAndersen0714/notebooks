# Spark Web UI 基础教程

## SQL

Spark Web UI 的 SQL 一栏中，展示了 Spark 执行计划对应的 SQL 视图，如果 Spark Application 中执行了 SQL，则会直接展示对应的 SQL 的实际执行计划在该页面。

同时 SQL 的实际执行计划最后的节点，也会显示当前 SQL query 每次执行的结果，如：写入操作，则执行计划的日志会显示写入的文件数和数据量。

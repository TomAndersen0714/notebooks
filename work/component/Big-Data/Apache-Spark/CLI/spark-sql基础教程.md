## 简介

Spark-sql
## 常用选项

`-d, --define <key=value>` ：定义 Spark SQL 客户端变量
`--queue` ：设置 Spark Application 提交队列
`--database` ：设置 Spark SQL 当前选择的数据库 
`-f <filename>` ：设置 Spark SQL 读取并执行的 SQL 文件名，SQL 文件中支持分号（semicolon）分割多个 SQL

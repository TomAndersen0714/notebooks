# MyBatisX 基础教程

MyBatisX 是一款 JetBrain Plugin，支持 MyBatis XML 文件与 Java Mapper 之间相关关联和映射，同时也支持 MyBatis Generator 自动生成相关的 MyBatis 代码（如：Mapper、Entity 等）和对应的 XML 配置文件，是 MyBatis 框架使用的必备插件。

# URI和URL的区别

简单理解：URL（Uniform Resource Locator）是网址类型的URI（Uniform Resource Identifier），相当于URI的子类，其访问和获取的是html，而URI是资源定位符，前缀可以是`file://, psql://, mongo://`等，其中包含有URL的`http://, https://`的前缀

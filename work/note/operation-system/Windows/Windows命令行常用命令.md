# Windows命令行常用命令


## cmd Shell

cd：打开指定路径

del：删除文件

where：查看可执行文件位置

echo：打印指定内容，如打印环境变量：`echo %GOPROXY%`


## PowerShell

\$env：打印环境变量，`$env:VariableName`，如：`$env:GOPROXY`

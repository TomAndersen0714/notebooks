## GoLand

安装完GoLang环境之后，GoLand基本就是开箱即用，无需特殊配置。

### 安装插件

#### .ignore

用于交互式生成.ignore文件

#### translation

一个JetBrain上的翻译插件，支持各大主流翻译工具，建议使用有道翻译


### 修改配置

#### Imports

File | Settings | Go | Imports

取消勾选“Add unambiguous imports on the fly”，避免GoLand每次调整代码时都会自动import module。

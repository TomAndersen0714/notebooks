# 体脂率计算公式

成年女性的体脂率计算公式
1. 参数 a=腰围 (厘米) x 0.74
2. 参数 b=体重 (千克) x 0.082 + 34.89
3. 身体脂肪总重量 (千克)= a - b
4. 体脂率=(身体脂肪总重量÷体重) x 100%

成年男性的体脂率计算公式
1. 参数 a= 腰围 (厘米) x 0.74
2. 参数 b=体重 (千克) x 0.082 + 44.74
3. 身体脂肪总重量 (千克) = a - b
4. 体脂率=(身体脂肪总重量÷体重) x 100%

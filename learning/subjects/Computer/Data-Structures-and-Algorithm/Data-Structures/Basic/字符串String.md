# 字符串 (String)


## Zigzag 转换

[LeetCode 6. Zigzag Conversion](https://leetcode.com/problems/zigzag-conversion/)

问题概述：将输入的字符串，按照指定行数，转换成 Z 字型，并返回字符按行拼接的结果。

解题思路：
1. Line by line，step by step。
	1. 时间复杂度： $O(n)$，空间复杂度：$O(n)$。

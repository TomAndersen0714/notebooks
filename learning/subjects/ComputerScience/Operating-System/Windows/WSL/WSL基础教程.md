# WSL 基础教程


## 安装

1. 检查 Windows 版本，要求版本号大于 1903
2. 设置-应用与功能-更多功能，启用“适用于 Linux 的 Windows 子系统”
3. Microsoft Store 搜索并安装 Ubuntu
4. Windows Terminal（终端）直接启动即可

参考：[知乎-史上最全的 WSL 安装教程](https://zhuanlan.zhihu.com/p/386590591)

# 数学常用符号


## 希腊字母（Greek Letter）

| 英文单词 | 大写形式   | 小写形式   |
| -------- | ---------- | ---------- |
| Alpha    | $A$        | $\alpha$   |
| Beta     | $B$        | $\beta$    |
| Gamma    | $\Gamma$   | $\gamma$   |
| Delta    | $\Delta$   | $\delta$   |
| Epsilon  | $E$        | $\epsilon$ |
| Zeta     | $Z$        | $\zeta$    |
| Eta      | $H$        | $\eta$     |
| Theta    | $\Theta$   | $\theta$   |
| Iota     | $I$        | $\iota$    |
| Kappa    | $K$        | $\kappa$   |
| Lambda   | $\Lambda$  | $\lambda$  |
| Mu       | $M$        | $\mu$      |
| Nu       | $N$        | $\nu$      |
| Xi       | $\Xi$      | $\xi$      |
| Omicron  | $O$        | $\omicron$ |
| Pi       | $\Pi$      | $\pi$      |
| Rho      | $P$        | $\rho$     |
| Sigma    | $\Sigma$   | $\sigma$   |
| Tau      | $T$        | $\tau$     |
| Upsilon  | $\Upsilon$ | $\upsilon$ |
| Phi      | $\Phi$     | $\phi$     |
| Chi      | $X$        | $\chi$     |
| Psi      | $\Psi$     | $\psi$     |
| Omega    | $\Omega$   | $\omega$   |
